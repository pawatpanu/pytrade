Optional branding assets for Inno Setup installer.

Place files in this folder before compiling installer\PyTradeSetup.iss:

1) setup.ico
   - Installer EXE icon.

2) wizard.bmp
   - Left wizard image (recommended 164x314 or similar Inno Setup style BMP).

3) wizard_small.bmp
   - Top-right wizard image for modern style (recommended 55x55 BMP).

If these files are not present, the installer will still compile and use default visuals.
